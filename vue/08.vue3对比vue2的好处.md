## Vue3 对比 Vue2 的好处

### 性能更好

优化点                Vue2                                   Vue3

虚拟DOM               基于对象 diff，性能一般                    基于 Proxy + 静态标记（静态提升、Block Tree）更快

响应式系统             Object.defineProperty（只能监听已有属性）  Proxy（可监听对象新增、删除、数组下标等）

打包体积               Vue core ≈ 20KB+                        Vue core < 10KB（tree-shaking 更好）

组件初始化             每次全量更新                               编译时标记静态节点，只更新变化的部分


### composition Api

Vue 2 用的是 Options API（data、methods、computed、watch...），而 Vue 3 新增了 Composition API（setup、ref、reactive、watchEffect...）

1.逻辑更清晰：把相同功能的逻辑放在一起，而不是分散在不同选项里。


2.更易复用：通过封装自定义 hook（例如 useUser(), useFetch()）实现逻辑复用。

3.更好地支持 TypeScript：Composition API 原生支持类型推导
