## 3. 作用域插槽
子组件的数据传递给父组件
### 子组件

<template>
  <ul>
    <li v-for="user in users" :key="user.id">
      <!-- 把 user 传给父组件 -->
      <slot :user="user">{{ user.name }}</slot>
    </li>
  </ul>
</template>

<script setup>
const users = [
  { id: 1, name: '小明' },
  { id: 2, name: '小红' }
]
</script>

### 父组件
```
<template>
  <UserList>
    <template #default="{ user }">
      <strong>{{ user.name }}</strong> (ID: {{ user.id }})
    </template>
  </UserList>
</template>
```
