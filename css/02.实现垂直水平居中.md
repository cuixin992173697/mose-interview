## 前端垂直水平居中的几种方案

```
<div class="parent">
	<div class="child">child</div>
</div>
```

### 1. 使用 position: absolute 实现

```
.parent {
	position: relative;
	width: 400px;
	height: 400px;
	background-color: red;
}

.child {
	position: absolute;
	width: 100px;
	height: 100px;
	background-color: aquamarine;
	left: 50%;
	top: 50%;
	transform: translate(-50%, -50%);
}
```

### 2. 使用 flex 实现

```
.parent {
	display: flex;
	justify-content: center;
	align-items: center;
	width: 400px;
	height: 400px;
	background-color: red;
}

.child {
	width: 100px;
	height: 100px;
	background-color: aquamarine;
}
```

### 3.使用 margin: auto

```
.parent {
	position: relative;
	width: 400px;
	height: 400px;
	background-color: red;
}

.child {
	position: absolute;
	width: 100px;
	height: 100px;
	top: 0;
	right: 0;
	bottom: 0;
	left: 0;
	margin: auto auto;
	background-color: aquamarine;
}
```

### 4.使用 grid 布局

```
.parent {
	display: grid;
	place-items: center;
	width: 400px;
	height: 400px;
	background-color: red;
}

.child {
	width: 100px;
	height: 100px;
	background-color: aquamarine;
}
```

### 5. 行内元素使用 vertical-align

```
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Document</title>
    <style>
      .box {
        display: inline-block; /* 变为行内块元素 */
        width: 100px;
        height: 100px;
        line-height: 100px;
        text-align: center; /* 文字水平居中 */
        background-color: aquamarine;
        vertical-align: middle; /* 与文字中线对齐 */
      }
    </style>
  </head>
  <body>
    <div class="box">box内容</div>
  </body>
</html>
```
