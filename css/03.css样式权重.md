### css样式权重

| 类型                        | 示例                              | 权重   |
| ---------------            | -------------------------------   | ---- |
| 行内样式                     | `<div style="color:red">`        | 1000 |
| ID 选择器                    | `#app`                           | 0100 |
| 类/伪类/属性选择器             | `.box`, `:hover`, `[id="main"]`  | 0010 |
| 元素/伪元素选择器             | `div`, `p::after`                 | 0001 |
| 通配符 / 继承 / 默认样式       | `*`, 继承                         | 0000 |
| `!important`                | 特殊规则，最高优先                  |      |
