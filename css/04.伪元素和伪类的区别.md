### 伪元素和伪类的区别

| 名称      | 作用                             | 语法示例                      | 目标对象              |
| -------  | ------------------               | -------------------------   | ----------------- |
| **伪类**  | 选择元素的某种状态或位置             | `:hover`, `:nth-child(2)`   | 已存在的元素状态或位置       |
| **伪元素** | 生成一个虚拟的子元素，插入到元素内部  | `::before`, `::after`       | 元素的某个“虚拟部分”或创建新内容 |


伪元素不在 HTML 里，但在渲染时出现。
