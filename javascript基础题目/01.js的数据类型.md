## 1.js的数据类型

### 1.1 基本数据类型

Number String Boolean Undefined Null Symbol Bigint

### 1.2 引用数据类型

Object

### 1.3 判断数据类型

typeof 不能判断Object类型

instanceof 判断某个对象是否由某个构造函数创建（只能用于引用类型）。 只能判断Object类型，对基本类型无效

Object.prototype.toString.call()
常用封装Object.prototype.toString.call(value).slice(8, -1)
